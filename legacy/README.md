# Legacy — drop your code slice here

This folder is **empty on day one**. It exists for [Extension 04 — Legacy Modernization](../docs/extensions/04_legacy_modernization/).

When you start Extension 04:
1. Drop a single legacy code slice (e.g., `SampleController.cs`, `Web.config`, a `.csproj`) here.
2. Follow the Copilot-driven modernization walkthrough in `docs/extensions/04_legacy_modernization/README.md`.
3. Your modernized service lands at `apps/<service_name>/`.
4. Then complete [Extension 05](../docs/extensions/05_wire_legacy_to_agent/) to wire it as a tool on a specialist.

No real MTA source code should land here — use synthetic slices only.
